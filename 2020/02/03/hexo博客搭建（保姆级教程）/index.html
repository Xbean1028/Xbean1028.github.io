# hexo博客搭建（保姆级教程）

环境准备：win10，hexo，github，node.js

效果展示： https://xbean1028.github.io/

  1. 部署环境

    1. 下载安装Node.js

    2. 到cmd中

    3. 新建一个文件夹，并进入该路径

    5. 说明：

      \source\ _posts目录下放置md文件夹，可以手动编辑，添加。博客内容makedown格式。

  2. 部署GitHub

    1. 登录GitHub

    2. 新建仓库，仓库名称必须为”名称.github.io”。（例如：我的GitHub：https://github.com/Xbean1028，所以我的仓库为Xbean1028.github.io ）

    3. 在博客目录下安装Git插件

    4. 设置 -config.yml文件
      最底部修改

      #Deployment

      #Docs: https://hexo.io/docs/deployment.html

      repo: 你的仓库地址（冒号后面有空格）

      branch: master

    5. 部署到远端

      根据提示输入账号密码

      部署完可能有延迟，等一会就好了

    6. 更换主题

      可以百度找，这里提供一个例子，下载下来放theme文件夹

      github.com/litten/hexo-theme-yilia

      设置 -config.yml文件，修改一行

  1. 通过Xbean1028.github.io 访问博客网站

    搞定，尽情享用吧
