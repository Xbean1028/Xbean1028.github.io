git命令使用简单实用

本文主要是初学者第一次食用

  1. 在GitHub新建一个仓库，这个很简单，不细说了

  2. 把仓库clone到本地，找个位置放就行

    git clone https://github.com/Xbean1028/XXXX.git

  3. 初始化
    git init

  4. 设置用户名和邮箱(全局)

  1. 将该文件添加到暂存区
    git add .
  2. 提交到本地版本库中
    git commit -m
    附：
    1)-m后跟注释内容
    2)可以 直接 在后面跟上文件，表示把该文件提交到本地版本库（不推荐）
    3)不跟任何文件名称，表示把当前所有暂缓区中的内容都提交到 本地版本库 中
    下面两种写法都可以，这里用的后一中
  1. 删除文件

  1. git config –list 查看设置过的参数

  2. 执行git push,上传到远程仓库
